# Project Title: প্রত্যর্পণ

## Overview
"প্রত্যর্পণ" is a full-stack web application designed as a charity platform focused on distributing clothes and supporting children in need. The application allows users to register, donate, and view a gallery of items available for distribution. It also includes an admin dashboard for managing donations and users.

## Features
- User Registration: Users can create accounts to participate in donations and view the gallery.
- Donation System: Users can make donations through a secure payment gateway.
- Admin Dashboard: Admin users can manage donations, users, and gallery items.
- Gallery Section: A dedicated section to display images of items available for distribution.

## Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **Payment Processing**: Razorpay
- **Authentication**: Google OAuth and email/password

## Setup Instructions

### Backend
1. Navigate to the `backend` directory.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm start
   ```
